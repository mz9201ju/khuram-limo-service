# KLimo Service — README

## Project overview
Limo Service is a responsive, business-focused website for a luxury ground-transport service.  
Built as a modern single-page (or multi-page) web app to showcase services, fleet, booking/contact flow, and business info. Designed to be production-ready and SEO-friendly.

## Key features
- Responsive UI (desktop → mobile)
- Fleet gallery and vehicle detail modal
- Booking / contact form (frontend hookup to API)
- Hero with CTA, services section, pricing cards
- Tailwind CSS + utility-first styling
- Basic SEO meta + sitemap/robots support

## Tech stack
- Frontend: **React** (Vite)
- Styling: **Tailwind CSS** + PostCSS
- Optional backend: **Node.js / Express** (or any REST API)
- Build & deploy: Vite build, deploy to Vercel / Netlify / GitHub Pages
- Dev tooling: npm (or pnpm / yarn), ESLint, Prettier
